# Gamenative-SteamGridDb-setter

## How to use

1. get SteamGridDb Key from (link)[https://www.steamgriddb.com/profile/preferences]

2. Insert This Key to /sdcard/steamgriddb_key.txt file (Create This file if there isn't)

3. install this apk and activate either LSPOSED (Zygisk) or LSPATCH (rootless but requires Gamenative patch via Lspatch)
